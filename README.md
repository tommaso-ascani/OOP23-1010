<p align="center">Our project is a replica of the well-known game 1010.</p>
<p align="center">The goal is to score as many points as possible by placing randomly generated blocks on a grid.</p>

# Features
- Sound control
- Language selection
- Grid size selection
- Theme shop
- Game saving
- Game resume
- Best score
